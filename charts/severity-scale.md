## Vertical Chart (for easel)

```
  ▲ HIGH
  │   • Permanent harm
  │   • Reputation damage  
  │   • Physical injury
  │   • "Would I do this with a gun?"
  │
  │ MEDIUM
  │   • Temporary pain
  │   • Recoverable
  │   • Touch/intimacy mistakes
  │   • Emotional hurt
  │
  │ LOW
  │   • Inconvenience
  │   • Awkwardness
  │   • No real harm
  │   • Miscommunication
  ▼
```

## Simple Version (for quick reference)

```
┌─────────────────────────────────────┐
│  HIGH    Permanent / Gun territory  │
├─────────────────────────────────────┤
│  MEDIUM  Temporary / Recoverable    │
├─────────────────────────────────────┤
│  LOW     Inconvenient / No harm     │
└─────────────────────────────────────┘
```

## Key Teaching Point

**Your response should match this scale.**

HIGH mistake → HIGH response (appropriate)
LOW mistake → HIGH response (YOU become the problem)
