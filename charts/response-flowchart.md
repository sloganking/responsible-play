## When a Mistake Happens

```
            MISTAKE OCCURS
                  │
                  ▼
        ┌─────────────────┐
        │     PAUSE       │
        │     BREATHE     │  ← Before reacting
        └────────┬────────┘
                 │
                 ▼
        ┌─────────────────┐
        │  What is the    │
        │  SEVERITY?      │────────────┐
        └────────┬────────┘            │
                 │                     ▼
                 │              ┌─────────────┐
                 │              │ LOW         │
                 │              │ MEDIUM      │
                 │              │ HIGH        │
                 │              └─────────────┘
                 ▼
        ┌─────────────────┐
        │  What is the    │
        │  TYPE?          │────────────┐
        └────────┬────────┘            │
                 │                     ▼
                 │              ┌─────────────┐
                 │              │ Malicious?  │
                 │              │ Accident?   │
                 │              │ Fawning?    │
                 │              │ Other?      │
                 │              └─────────────┘
                 ▼
        ┌─────────────────┐
        │  Is this about  │
        │  NOW or my PAST?│  ← Check your filters
        └────────┬────────┘
                 │
                 ▼
        ┌─────────────────┐
        │  MATCH RESPONSE │
        │  TO SEVERITY    │
        └────────┬────────┘
                 │
                 ▼
        ┌─────────────────┐
        │  REPAIR         │
        │  not punishment │
        └─────────────────┘
```

## Simplified Version (for easel)

```
1. PAUSE
2. SEVERITY? (Low / Medium / High)
3. TYPE? (Malicious / Accident / Fawning)
4. NOW or PAST? (Check your trauma)
5. MATCH RESPONSE to SEVERITY
6. AIM for REPAIR
```

## Warning Signs You're Over-Responding

- You want to hurt them back
- You want others to take your side
- You're treating an accident like malice
- Your response would cause more harm than their mistake did
