## The Drama Triangle (Dysfunctional)

```
                         PERSECUTOR
                        "It's your fault"
                        Blames, attacks
                              ▲
                             / \
                            /   \
                           /     \
                          /       \
                         /         \
                        /           \
                       /             \
            VICTIM ◄─────────────────► RESCUER
         "Poor me"                    "Let me save you"
         Helpless                     Enables
         Waits                        Fixes

                    ↻ ROLES ROTATE ↺
```

## The Empowerment Dynamic (Healthy)

```
                         CHALLENGER
                        "I'll push you to grow"
                        Accountability + care
                              ▲
                             / \
                            /   \
                           /     \
                          /       \
                         /         \
                        /           \
                       /             \
           CREATOR ◄─────────────────► COACH
         "What do I want?"           "How can I support?"
         Owns power                  Empowers
         Creates outcomes            Asks questions

                    ✓ STABLE ROLES ✓
```

## For Easel (Simplified)

```
DRAMA:                          HEALTHY:

    Persecutor                      Challenger
         ▲                              ▲
        / \                            / \
       /   \                          /   \
  Victim ── Rescuer             Creator ── Coach

  (roles rotate)                (roles stable)
  (reactive)                    (chosen)
  (stuck)                       (empowered)
```

## Key Teaching Points

1. **Drama roles are reactive** — you fall into them unconsciously
2. **Roles rotate** — Victim becomes Persecutor, Rescuer becomes Victim, etc.
3. **Empowerment roles are chosen** — you step into them consciously
4. **The key difference is agency** — Creator, Challenger, Coach all have power
5. **In intimacy spaces** — watch for Rescuers who judge without info, Persecutors who mob, Victims who won't take responsibility
